# letをvarに変えた場合
## 予想
- 0から10までの数字が出力される
- varはletと異なりブロックスコープではないため、forループの外のスコープからも参照できるため、最後のインクリメントが反映されてループが抜けたあとのi:10が表示される
- また、functionの中で再宣言されているvar iは関数スコープとして扱われるため、forループで表示に使われるiには影響を与えない

- 予想通りの結果となった
